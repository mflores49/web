🗺️ Curso
Creación de Mapas Web Interactivos con Tecnologías Open Source

👉 ¿Te gustaría transformar datos geográficos en mapas interactivos accesibles desde cualquier parte del mundo? Este curso está diseñado para profesionales, estudiantes y emprendedores que buscan visualizar, analizar y compartir información espacial en línea de forma moderna, eficiente y totalmente libre de licencias privativas.

🎯 ¿Qué aprenderás?
Domina el flujo completo básico y simple de creación de mapas web con herramientas de código abierto:

✅ QGIS: Prepara, edita y gestiona datos geoespaciales con precisión.
✅ Leaflet: Diseña mapas interactivos profesionales que se adaptan a cualquier dispositivo.
✅ Visual Studio Code: Personaliza y programa funcionalidades básicas con HTML, CSS y JavaScript.
✅ GitHub Pages: Publica tus proyectos públicos en línea, de forma gratuita y segura.

🌐¿Por qué elegir tecnologías open source?
💡Decisiones basadas en datos → Convierte bases de datos y archivos GIS en mapas interactivos que comunican resultados de forma clara y visual.
💸Reducción de costos → Aprende a utilizar herramientas gratuitas y escalables, sin depender de software privativo.
🔓Autonomía tecnológica → Desarrolla y publica tus propios proyectos sin intermediarios, mostrando resultados en tiempo real a clientes, colegas o ciudadanía.
